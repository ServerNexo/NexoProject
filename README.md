🛡️ Nexo Network - Modular System
Este es un sistema de microservicios para Minecraft 1.21.11.

📦 Estructura:
NexoCore: Cerebro y API de base de datos (Supabase/Hikari).

NexoItems: Gestión de RPG e items custom.

NexoPvP: Sistema de combate y pasivas.

NexoColecciones: Progresión de farmeo y Slayers.

NexoMinions: Automatización.

🛠️ Compilación:
Ejecutar mvn clean install en la raíz para generar todos los .jar automáticamente.
